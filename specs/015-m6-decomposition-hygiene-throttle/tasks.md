## Ordering principle

M6 extends a shipped M5.x substrate; the schema additions (T001) come first because every package built on top references the new columns or the new table. The ARCHITECTURE.md amendment (T019) lands AFTER implementation rather than before — different from M5.4's amendment-first discipline because M6's amendment is annotative ("M6 shipped: X, Y, Z" + schema additions in the documented schema block) rather than spec-shaping; landing it after the substrate exists keeps the substring-pinned text honest.

The `internal/throttle/` package (T004 + T005) lands before any caller invokes it, so the gate predicates can be tested in isolation against fakes and a testcontainer Postgres. The cost-telemetry result-grace (T006) is independent — `internal/spawn/pipeline.go` extension that closes the cost-telemetry blind spot — and lands before T007 which depends on honest cost telemetry for the budget gate to be meaningful.

T007 wires the spawn-prep gate; T008 + T009 wire the rate-limit actuator and its chaos test. Both depend on the throttle package + the new schema.

T010 (decomposition verb) and T011 (per-turn ticket cap) are independent of throttle; they could land earlier in the sequence, but they're scheduled after the throttle plumbing so the integration test (T018) exercises decomposition + budget defer + rate-limit pause in one chat session.

T012 (hygiene predicates) is independent of every other thread — extends `internal/hygiene/evaluator.go` only. Lands once the new `thin_diary` status is referenceable from sqlc (already satisfied by T001 + T002).

T013 + T014 wire the new env vars and supervisor lifecycle composition. T015 → T017 build the dashboard surface on top of the supervisor work.

T018 is the golden-path integration test — chat → CEO decomposes → 3 tickets → engineer spawns → live throttle event observed. T019 lands the ARCHITECTURE.md amendment + test pin. T020 walks the spec's success criteria as a scripted check; T021 ships the retro.

Zero new Go dependencies. Zero new TypeScript dependencies. Locked-deps streak continues past M5.4 → M6.
